# Olfactory Profile Specification

TSEL does not define a separate olfactory schema. Each olfactory class maps into the shared TSEL contract and records its class in `contextual_metadata.domain_profile`.

| Profile ID | Class Name | Expected Temporal Structure | Expected Input Structure | Minimum Required Metadata | Optional Metadata | Supported TSEL Claims | Forbidden Claims | Likely Unresolved States |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| `olfactory_event_profile` | Olfactory exposure/event logs | Sparse timestamped observations or markers | Row-style CSV or JSON with odor or event context | timestamp or relative time, source, odor or measurement field | odor identity, concentration, trial ID, presentation phase, delivery state | observation events, marker events, explicit stimulus context, explicit concentration context | implicit continuity, invented odor identity, continuous sample claims without samples | missing odor identity, missing concentration, missing onset/offset markers, missing trial ID |
| `olfactory_sensor_stream_profile` | Olfactory sensor streams | Timestamped or sequence-timed numeric samples | Multichannel sensor CSV or JSON | time basis, source, numeric channels | sampling rate, odor identity, concentration, trial ID, acquisition metadata | sample streams, deterministic continuity, deterministic phase recovery when explicit markers support it | subjective-report inference, invented device metadata, invented stimulus meaning | missing sampling rate, missing odor identity, missing concentration, missing trial ID |
| `olfactory_receptor_series_profile` | Olfactory receptor-response series | Timestamped or sequence-timed receptor-aligned series | Tabular or JSON receptor series with numeric channels | time basis, source, receptor identifier, numeric channels | sampling rate, odor identity, concentration, trial ID | sample streams, receptor-series identity, deterministic continuity | invented neural route, invented subjective context, invented odor identity | missing receptor identifier, missing sampling rate, missing odor identity, missing trial ID |
| `olfactory_neural_response_profile` | Olfactory neural-response time series | Sampled neural traces aligned to odor events | EEG-like multichannel series plus odor markers or odor trial context | neural channels, time basis, odor marker or olfactory trial context | sampling rate, odor identity, concentration, trial ID, pre/post windows | EEG acquisition route, sample streams, deterministic continuity, deterministic phase recovery when markers support it | implicit olfactory domain without odor evidence, invented stimulus context, invented trial structure | missing odor markers, missing odor identity, missing concentration, missing trial ID |
| `olfactory_trial_packet_profile` | Olfactory trial packets | Packet-derived observations, vectors, or markers | Typed packet directory or declared packet collection | packet membership, packet type | absolute time, trial ID, odor identity, concentration | packet-declared profile identity, packet-declared trial context, packet-declared partition context | invented absolute time, invented acquisition device | missing absolute time, incomplete packet declaration |
| `olfactory_subjective_report_profile` | Olfactory subjective report streams | Sparse timestamped or sequence-timed report observations | Rows or JSON records with text or subjective ratings linked to odor context | source, report field, odor or trial context | timestamp, odor identity, trial ID, concentration | report events, explicit report context, explicit odor context | sample continuity, neural route, invented stimulus window | missing odor identity, missing timestamp, missing trial ID |

## Mapping Rules
- Every profile still maps into the same seven-field top-level event schema.
- Profile identity is stored in `contextual_metadata.domain_profile.profile_id`.
- Olfaction as a sense is stored in `contextual_metadata.sensory.primary_sense`.
- Acquisition route is stored in `contextual_metadata.acquisition.acquisition_profile` and may differ from the olfactory sense itself, for example `eeg` for olfactory neural-response recordings.
- Missing or weak evidence is exposed through `contextual_metadata.completeness` and `contextual_metadata.unresolved` rather than guessed.
